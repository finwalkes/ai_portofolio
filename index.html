A. Frontend (Client-side)

  • Core: HTML, CSS, JS (ES6+)
  • Framework/Library: React, Vue, Angular, Svelte, Solid, Alpine
  • Styling: Tailwind, Bootstrap, Material UI, Styled Components, Emotion, Chakra UI
  • Build Tools: Vite, Webpack, Rollup, Parcel, Snowpack
  • SSR/SSG/SPA: Next.js, Nuxt.js, Gatsby, Astro, SvelteKit
  • State Management: Redux, Context API, MobX, Pinia, Zustand, Recoil
  • Testing: Jest, React Testing Library, Cypress, Playwright, Vitest
  • Animations: Framer Motion, GSAP, Anime.js
  • PWA: Service Worker, Workbox, Web Manifest
  • Accessibility: ARIA, Lighthouse, axe-core
